Neural Network

The Linear Regression models used earlier are very powerful, but excel at tasks like image identification and are not necessarily the best choice for time series data. The Python TensorFlow package offers options that might be better fit. Specifically, the Recurrent Neural Networks (RNN) are a class of neural networks for modeling time series data. RNN has 3 layers which can be utilize: the SimpleRNN, Gated Recurrent Units (GRU) and Long Short-Term Memory (LTSM). The LTSM layer is better suited for longer time series where information may be contained in time steps further away from the point being processed, as the loss function for the simpleRNN is exponential, so it does not retain information that far out. The GRU is similar to the LTSM, but is mainly more efficient and not more accurate.
